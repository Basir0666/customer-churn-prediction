# Customer Churn Prediction

## Overview

This project focuses on analyzing customer data to understand why customers leave a service and to build a model that can predict churn. The workflow covers data cleaning, exploratory analysis, and machine learning.

---

## What I Did

- Loaded and explored the dataset to understand its structure
- Performed exploratory data analysis to identify patterns related to churn
- Cleaned the data by handling missing values and removing irrelevant columns
- Converted categorical variables into numerical format
- Scaled numerical features for better model performance
- Built a Random Forest model to predict customer churn
- Evaluated the model using accuracy and classification metrics

---

## Key Findings

- Customers with shorter tenure are more likely to leave
- Higher monthly charges are associated with higher churn
- Month-to-month contract customers tend to churn more
- Certain payment methods (like electronic check) show higher churn behavior
- Long-term customers are more stable and less likely to churn
